# TufDB 💻

> Too Useful For *DataBreach* - v1

A simple interactive Python script that scan the internet to find your password.

## 🌴 Installation

```
git clone https://github.com/Fluff2513/TufDB
cd TufDB
pip install -r requirements.txt
python tufdb.py
```

## 🎀 Usage

```
python tufdb.py
```

Then just answer the questions - TufDB does the rest!
